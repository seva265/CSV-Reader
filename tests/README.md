# Тесты

## 📁 Структура тестов

```
tests/
├── conftest.py                 # Общие фикстуры для тестов
├── fixtures.py                 # Фикстуры для работы с базой данных
├── test_upload_errors.py       # Тесты загрузки CSV файлов
├── test_database_errors.py     # Тесты ошибок базы данных
└── test_analysis_endpoints.py  # Тесты аналитических эндпоинтов
```

## 🚀 Запуск тестов

### Быстрый запуск (рекомендуется)
```bash
python scripts/run_tests.py
```

### Через pytest напрямую
```bash
python -m pytest tests/ -v
```

### Через Poetry
```bash
poetry run pytest tests/ -v
```

## Что проверяется

### CSV загрузка (`test_upload_errors.py`)
- Отсутствие файла при загрузке
- Попытка загрузить не-CSV файл
- Неверные заголовки в CSV
- Некорректные оценки и даты в данных

### База данных (`test_database_errors.py`)
- Ошибки подключения к базе данных
- Сбои при выполнении SQL запросов

### Аналитические эндпоинты (`test_analysis_endpoints.py`)
- Получение пустых результатов
- Обработка данных с результатами
- Работа обоих эндпоинтов анализа

## 📊 Покрытие кода

Тесты покрывают основные сценарии ошибок и успешной работы:
- Валидация входных данных
- Обработка ошибок базы данных
- Корректная работа аналитических функций

## 🔧 Конфигурация

- `conftest.py` - фикстуры и общие настройки тестов
- `fixtures.py` - фикстуры для работы с реальной базой данных (используются при USE_REAL_DB=1)

## 📝 Зависимости

Для тестирования используются:
- `pytest` - фреймворк для тестирования
- `pytest-asyncio` - поддержка асинхронных тестов

Зависимости указаны в `pyproject.toml` в секции `[tool.poetry.group.dev.dependencies]`.
